# Relax Beauty Zone — Website

A premium, mobile-responsive landing page for **Relax Beauty Zone**, a ladies-only beauty parlour in Dombivli East, Thane.

**Live site:** https://appdeloper.github.io/relaxbeautyzone/

## Features
- Fully responsive (mobile, tablet, desktop)
- WhatsApp-powered booking form + click-to-chat buttons
- Full service menu (Facials & Hydrafacial, Hair, Makeup, Bridal, Mehndi, Waxing, Threading, Mani-Pedi, Body Spa)
- Packages, reviews, FAQ accordion, Google Map & contact details
- Custom branding with the salon logo

## Tech
Single self-contained `index.html` (HTML + CSS + JS). No build step.

## Edit
Open `index.html` in any editor. Replace `logo.png` / `salon-interior.jpg` to update images.
